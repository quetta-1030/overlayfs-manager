# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of OverlayFS Manager
- Interactive TUI installer
- CLI commands: install, status, reset, maintain, uninstall
- Node.js API for programmatic usage
- systemd service integration
- Support for CentOS 7 and compatible distributions

### Changed
- Renamed from internal project to generic "OverlayFS Manager"
- Removed all vendor-specific references
